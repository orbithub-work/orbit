# 智归档 OS - 商业策略

> 最后更新：2026-02-19
> 定位：独立开发者路线

---

## 🎯 核心策略

```
Year 1-2：国内个人用户 → 验证产品 + 建立口碑 + 活下来
Year 3+：  国际化 → 规模化增长

企业用户：有人主动找再说，不主动推
```

## 💰 商业模式

### **Freemium + 插件生态**

#### **免费版**
```

项目管理：3 个项目
成品管理：10 个成品
数据看板：基础统计（仅本月数据）

目标：
- 让用户体验完整功能
- 建立使用习惯
- 降低决策门槛
```

#### **Pro 版（买断）**
```
价格：249 元（买断）
早鸟价：149 元（前 1000 名）

解除所有限制：
✅ 无限素材
✅ 无限项目
✅ 无限成品
✅ 完整数据看板（历史数据、对比分析）
✅ 高级功能（批量操作、智能标签等）

不包含插件
```

#### **插件（单独售卖）**
```
核心插件：
- Dock 悬浮窗：99 元 ⭐ 优先开发
- 全局快捷键：49 元
- AI Prompt 管理：79 元

工具集成：
- 剪映助手：69 元
- Premiere 助手：69 元
- Photoshop 助手：69 元

数据同步：
- 平台数据同步：49 元/平台

插件套餐：
- 工作流套餐：199 元（Dock + 快捷键 + AI）
  单买 227 元，省 28 元
```

## 🎯 关键策略

### **1. 聚焦个人用户**
```
✅ 产品简单易用
✅ 自助服务为主
✅ 社区互助
✅ 文档完善

避免：
❌ 复杂的企业功能
❌ 一对一售前咨询
❌ 定制开发
❌ 过度承诺
```

### **2. 国内练兵，国外收割**
```
Year 1-2：
- 国内市场验证产品
- 打磨功能
- 建立口碑
- 积累经验

Year 3+：
- 带着成熟产品去国际市场
- 降维打击
- 规模化增长
```

### **3. 企业用户：被动接单**
```
如果有企业主动找你：
✅ 报价高一点：999-2999 元/年
✅ 说明是"个人版多授权"，不是企业版
✅ 不承诺定制开发
✅ 不承诺 SLA
✅ 能接就接，不能接就算

心态：
"企业用户是意外之喜，不是主线"
```

### **4. 保持独立开发者心态**
```
✅ 做自己能掌控的事
✅ 不追求虚荣指标
✅ 关注真实收入
✅ 享受创作过程

避免：
❌ 盲目扩张
❌ 分散精力
❌ 过早优化
```

## 💡 定价心理学

### **为什么 249 元？**
```
✅ 不到 250 元，心理门槛低
✅ 比 Eagle（199）贵 50 元，但功能多 3 倍
✅ 用户会觉得"值"
✅ 足够支撑开发成本
```

### **为什么早鸟价 149 元？**
```
✅ 制造紧迫感
✅ 快速获取种子用户
✅ 建立口碑
✅ 早鸟用户会觉得"赚到了"，传播动力强
```

### **为什么插件单独售卖？**
```
✅ 持续收入
✅ 用户按需购买
✅ 降低 Pro 版价格门槛
✅ 提升客单价（套餐）
```

## ⚠️ 风险与应对

### **风险 1：用户增长慢**
```
应对：
- 加大内容营销投入
- 优化产品体验
- 降低早鸟价（99 元）
- 延长早鸟期
```

### **风险 2：付费转化率低**
```
应对：
- 优化免费版限制（更合理）
- 增加 Pro 版价值感
- 用户案例展示
- 限时优惠活动
```

### **风险 3：竞品抄袭**
```
应对：
- 快速迭代
- 建立品牌护城河
- 深耕用户社群
- 插件生态（难以复制）
```

### **风险 4：国际化失败**
```
应对：
- Year 1-2 在国内验证产品
- 充分准备（翻译、文档、支付）
- 小步试错（先上 Product Hunt）
- 持续优化
```
