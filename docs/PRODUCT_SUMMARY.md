## 🔥 五大核心亮点

### 1. 非侵入式素材管理
```
✅ 只索引，不移动文件
✅ 虚拟项目，绑定多源
✅ 指纹去重，百万级性能
✅ 血缘追踪，来源可溯

差异化：尊重用户文件习惯，不像 Eagle 强制导入
```

### 2. 模板化项目管理
```
✅ 预设模板（短视频/设计/游戏/摄影）
✅ 零学习成本，选模板即用
✅ 自动记录操作日志
✅ 素材-项目自动关联

差异化：降低门槛，用户无感知养成习惯
```

### 3. 插件化工作流辅助 ⭐ 核心护城河
```
✅ Dock 悬浮窗 - 拖入索引，拖出到剪映/PR/PS
✅ 全局快捷键 - 随时记录灵感/脚本/笔记
✅ AI Prompt 绑定 - 生成文件关联 Prompt，可搜索可复现
✅ 内网插件通信 - 独立进程，语言解耦，进程隔离

差异化：减少创作摩擦，这是别人抄不走的
```

### 4. 跨平台数据复盘
```
✅ 统一记录所有平台产出（抖音/B站/小红书/YouTube）
✅ 项目对比功能（类似汽车之家选车对比）
✅ 数据驱动洞察（最佳时间/高价值素材/效率分析）
✅ 素材使用频率统计

差异化：跨平台整合，平台自己做不到
```

### 5. 完整创作闭环
```
素材收集 → 项目创作 → 产出发布 → 数据复盘 → 优化迭代
     ↑                                              ↓
     └──────────────── 反馈循环 ─────────────────────┘

差异化：不是单点工具，是完整操作系统
```
